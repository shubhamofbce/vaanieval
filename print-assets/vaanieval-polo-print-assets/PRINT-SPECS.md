# VaaniEval white-melange polo print specifications

## Front chest

- Recommended print width: 90 mm
- Transparent PNG: 1063 × 650 px at 300 DPI
- Placement: wearer's left chest, approximately 70–90 mm below the shoulder seam

## Back

- Recommended print width: 280 mm
- Transparent PNG: 3307 × 3898 px at 300 DPI
- Placement: centered, with the artwork top approximately 75–90 mm below the collar seam

## Ink colors

- Charcoal: `#1F2828`
- Teal: `#0C706B`
- Mint: `#78CDB5`

The files have transparent backgrounds and no white artwork. The white-melange garment supplies the light areas.

Use the SVG files when the printer accepts vector artwork. Use the 300-DPI PNG files for raster workflows. Ask the printer to preserve transparency and the supplied physical dimensions.

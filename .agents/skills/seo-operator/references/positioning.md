# VaaniEval SEO positioning

- Audience: global English voice-AI agencies and teams operating production voice agents.
- Primary conversion: qualified design-partner application.
- Secondary conversion: GitHub repository visit or self-hosting adoption.
- Positioning: open-source, self-hosted QA workspace for production voice-agent conversations.
- Supported voice-provider claims: ElevenLabs conversation import, agent discovery, transcript/media review; Vapi adapter-based conversation import.
- Evaluation themes: task completion, resolution quality, unsupported claims, fallback behavior, latency, and operational signals.
- Avoid compliance guarantees, fabricated customer proof, invented benchmarks, and claims that all provider fields are universally available.
